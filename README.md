# Python Mini Projects

This repository contains beginner Python projects created to practice programming logic and basic Python concepts.

## Projects Included

### 1. Simple Calculator
A command-line calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

### 2. Number Guessing Game
The computer generates a random number between 1 and 100, and the user tries to guess it. The program gives hints like "higher" or "lower" until the correct number is guessed.

### 3. Student Marks Analyzer
Reads student marks and analyzes them to calculate statistics such as average marks, highest marks, and lowest marks.

### 4. To-Do List
A simple task manager where users can add, view, and remove tasks from their to-do list.

### 5. Contact Book
A basic contact management system that allows users to store and manage contact names and phone numbers.

### 6. CSV Data Analyzer
Reads data from a CSV file and performs basic analysis such as calculating averages or displaying stored records.

### 7. Snake Water Gun Game
A simple command-line game similar to Rock-Paper-Scissors where the player competes against the computer.

## Skills Practiced

- Python basics
- Conditional statements
- Loops
- Dictionaries and lists
- File handling
- CSV file processing
- Basic problem solving
